# Clima Prime Refrigeração

Landing page premium desenvolvida para uma empresa fictícia de refrigeração, com foco em conversão, geração de leads pelo WhatsApp e apresentação clara de serviços de instalação, higienização e manutenção de ar-condicionado.

Este projeto foi criado como peça de portfólio para demonstrar capacidade de desenvolver páginas comerciais modernas, responsivas e orientadas a resultado.

## Objetivo do projeto

A proposta da página é simular um site real para uma pequena empresa de serviços locais, com comunicação direta para residências, comércios e empresas.

O foco principal é transformar visitantes em contatos qualificados pelo WhatsApp, usando uma estrutura simples:

1. Apresentação clara da oferta.
2. Identificação das dores do cliente.
3. Serviços organizados em cards.
4. Oferta principal com redução de risco.
5. Explicação do processo de atendimento.
6. Prova social demonstrativa.
7. FAQ para remover objeções.
8. CTA final para orçamento.

## Como rodar o projeto localmente

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
````

Acesse a pasta do projeto:

```bash
cd NOME_DA_PASTA
```

Instale as dependências:

```bash
npm install
```

Rode o ambiente de desenvolvimento:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```

Visualize a build localmente:

```bash
npm run preview
```

## Deploy

O projeto está preparado para deploy no Netlify.

Configuração esperada:

```txt
Build command: npm run build
Publish directory: dist
```

O arquivo `netlify.toml` configura o build e o fallback para rotas internas da aplicação.
